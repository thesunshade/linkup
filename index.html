Specifications:

  • This tool will only create valid links if it is given valid citations
  • Chapter and sutta numbers should be separated by either a period or a semicolon. (e.g. SN 3.5, AN 6:2)
  • Ranges are not possible to link.
  • There is a slight variation between citations in the SN and AN for SuttaCentral.net and DhammaTalks.org (Ajahn Thanissaro). This tool assumes citations on SuttaCentral.net
  • SN = Saṁyutta Nikāya; Snp = Sutta Nipāta
  • The original text should not contain any linked citations or they will be double linked in the final output.
  • The text that you enter is never sent across the internet. All processing is done in your browser.
