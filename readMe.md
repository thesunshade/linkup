# Citation Link Up Tool

This tool allows you to imput a block of text and add links to the translation on SuttaCentral.net.

The following citations are allowed:

- DN Dīgha Nikāya
- MN Majjhima Nikāya
- SN Saṁyutta Nikāya
- AN Aṅguttara Nikāya
- Khp Khuddakapāṭha
- Dhp Dhammapada
- Ud Udāna
- Iti Itivuttaka
- Snp Sutta Nipāta
- Vv Vimānavatthu
- Pv Petavatthu
- Thag Theragātha
- Thig Therīgāthā
